# Loss Function (損失函數)

## 最小化    
在理想的模型中，我們希望預測值和實際值是相等的，但現實應用中預測值和實際值是有落差的    
而之間所謂的落差在統計上稱為殘差(residual)，實際應用上我們會希望殘差越小越好。

` loss/residual = y - ŷ `

y 表示實際值、 ŷ 表示預測值
